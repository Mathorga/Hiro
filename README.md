# hiro
## Commands
### enable (e)
#### char motor_select
l - left motor</br>
r - right motor</br>
a - all motors</br>
### disable (d)
#### char motor_select
l - left motor</br>
r - right motor</br>
a - all motors</br>
### run (r)
#### char motor_select
l - left motor</br>
r - right motor</br>
a - all motors</br>
#### int speed
-100 < speed < 100
