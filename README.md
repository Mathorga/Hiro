# hiro
## Commands
### enable (e)
#### params
##### char motor_select
l - left motor</br>
r - right motor</br>
a - all motors</br>
#### examples
Enable right motor:</br>
er
### disable (d)
#### params
##### char motor_select
l - left motor</br>
r - right motor</br>
a - all motors</br>
#### examples
Disable all motors:</br>
da
### run (r)
#### params
##### int motor_select
0 - left motor</br>
1 - right motor</br>
##### int speed
-100 < speed < 100
